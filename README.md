# Akan-Name-Generator.
An Akan-Name-Generator is a tool that assigns traditional Akan names based on the day of the week someone was born.

# Features.
-**Accurate day calculation.**
-**Input validation.**
-**Gender specific Names.**
-**Interactive display.**
-**Cultural information.**

# Demo.
Live Demo(#) <https://trizahn2002-source.github.io/Akan-Name-Generator./>

# Usage.
1. Open the application in your web browser.
2. Enter your birth date using the date picker.
3. Select your gender(Male or Female).
4. Click 'Get your Akan Name' to discover your Akan Name.
5. Your Akan Name will be displayed along with the day of the week your were born.

# How It Works.
The application uses the following algorith;
1. **User Input**: Accepts birthdate and gender from the user.
2. **Day Calculation**: Uses JavaScript's 'Date' object to determine the day of the week.
3. **Name Mapping**: Matches the day and the gender to the corresponding Akan Name.
4. **Display**:Shows the result to the user.
